Introducción

Para el proyecto final se desarrolló un robot capaz de realizar dibujos artísticos utilizando un brazo robótico UR5e. Este proyecto integra conocimientos de programación, control de robots y diseño mecánico para lograr un sistema que genera e interpreta imágenes para reproducirlas en papel mediante un marcador.

Reporte de proyecto

En el reporte se puede visualizar que se dividio en varias etapas, principalmente fueron:
- Diseño del circuito eléctrico.
- Programación del PLC S7-1200.
- Programación de la HMI KTP400.
- Programación del robot UR5e.
- Mejora de estructura de robot.
- Creación y monteje de platina metálica.
- Creación y montaje de botonera metálica.
- Limpieza y pintura de la estructura.
- Cambio de llantas.
- Creación de interfaz web para generación de imágenes con IA y su procesamiento.
- Creación de interfaz web para control del robot.
- Creación de interfaz web para control del PLC.

Conclusión

El proyecto permitió integrar los conocimientos adquiridos en clase con sistemas industriales de forma coordinada para lograr :

- Comunicación entre PC y Robot UR5e.
- Diseño avanzado y so de herramientas especializadas.
- Simulación en roboDK.
- Seguimineto de curvas precisas.
